Detailed Description

Interface for DFRobot's Power Shield (SKU:DFR0105)

See Also
http://www.dfrobot.com/wiki/index.php/Power_Shield_(SKU:DFR0105) for hardware details.

By default, the auxiliary ower output is disabled (see implementation of DFRobotPowerShield::init() ).
